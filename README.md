# 📊 Nassau Candy Profitability Dashboard

Interactive dashboard to analyze **product profitability and margin performance** using Streamlit and Plotly.

## 📄 Research Paper

---

## 🎯 Objective
“High Sales ≠ High Profit” — identify profitable products, low-margin risks, and division performance.

---

## 📊 Features
- KPI Metrics (Sales, Profit, Margin)
- Product Profitability Analysis
- Division Performance
- Cost vs Sales (Scatter)
- Pareto Analysis (80/20 Rule)
- Smart Filters (Date, Division, Product, Margin, Profit)

---

Python • Streamlit • Pandas • Plotly
